
# Formas de obtener un <input> con React Testing Library

1. getByPlaceholderText
   - Busca un input por su atributo `placeholder`
   - Ejemplo:
     <input placeholder="Ingrese su usuario" />
     screen.getByPlaceholderText(/usuario/i)

2. getByLabelText
   - Usa el texto de una etiqueta <label> asociada al input
   - Ejemplo:
     <label htmlFor="email">Correo</label>
     <input id="email" />
     screen.getByLabelText(/correo/i)

3. getByRole
   - Usa el rol accesible del input (ej: "textbox")
   - Más confiable cuando el input tiene label o aria-label
   - Ejemplo:
     <input type="text" aria-label="Usuario" />
     screen.getByRole('textbox', { name: /usuario/i })

4. getByAltText
   - Se usa para imágenes, no inputs. Busca por el atributo `alt`
   - Ejemplo:
     <img src="foto.jpg" alt="Foto de perfil" />
     screen.getByAltText(/foto de perfil/i)

5. getByTestId (último recurso)
   - Se usa cuando no hay otro selector visible o accesible
   - Ejemplo:
     <input data-testid="email-input" />
     screen.getByTestId("email-input")
